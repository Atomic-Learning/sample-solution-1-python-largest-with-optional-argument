Example solution using a loop and abs()

This version checks every entry in the list and tracks the largest absolute value manually. We begin by assuming the first element is the largest. We then proceed to loop over every other element in the list. If abs_value is True, we update the loop variable to be the absolute value of the current element.

def largest(list_in, abs_value=False): max_value = list_in[0] for value in list_in[1:]: if abs_value: value = abs(value) if value > max_value: max_value = value return max_value

Test calls

print(largest([1, 2, -3])) print(largest([1, 2, -3], False)) print(largest([1, 2, -3], True)) print(largest([1, 2, -3, 4], True))

Discussion

This function is guaranteed to work correctly for all non-empty lists and both values of abs_value. However, the explicit for-loop, which includes two conditionals checks is not particularly efficient for large lists and alternative approaches might be considered for better performance.
